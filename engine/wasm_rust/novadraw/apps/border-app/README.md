# Border App - Border 装饰器验证

## 功能说明

验证 Draw2D Border 装饰器系统，包括各种边框样式、圆角、组合边框等效果。

## 运行方式

```bash
cargo run -p border-app
```

## 场景说明

| 场景 | 名称 | 验证内容 |
|------|------|----------|
| 0 | RectangleBorder 基本 | 不同宽度和颜色的边框 |
| 1 | 圆角边框 | 带圆角效果的边框 |
| 2 | 实线边框 | 不同粗细的实线边框 |
| 3 | 边框+背景 | 边框与背景色的组合 |
| 4 | 嵌套 Border | 多层嵌套的边框效果 |
| 5 | 边框动画 | 动态变化的边框 |
| 6 | 动态边框 | 可交互修改的边框 |
| 7 | 组合 Border | 多重边框叠加 |
| 8 | 内边距 | 边框内边的间距效果 |
| 9 | 特殊边框 | 渐变、多色等特殊边框 |

## 操作说明

- 按数字键 `0`-`9` 切换场景
- 按 `ESC` 退出程序

## Border 类型

### LineBorder

实线边框，支持不同的颜色和宽度。

### RectangleBorder

矩形边框，支持圆角处理。

### 组合边框

多个边框叠加形成复合效果。

## 边框属性

| 属性 | 说明 |
|------|------|
| 颜色 | 边框线的颜色 |
| 宽度 | 边框线的粗细 |
| 圆角 | 边框角的弧度 |
| 样式 | 实线、虚线等 |

## 依赖模块

- `novadraw-scene`: 场景图和 Figure 接口
- `novadraw-render`: Vello 渲染后端
