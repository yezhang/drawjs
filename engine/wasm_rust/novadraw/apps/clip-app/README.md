# Clip App - 裁剪机制验证

## 功能说明

验证 Draw2D 裁剪系统，包括基础裁剪、嵌套裁剪、多层裁剪等机制。

## 运行方式

```bash
cargo run -p clip-app
```

## 场景说明

| 场景 | 名称 | 验证内容 |
|------|------|----------|
| 0 | 基础裁剪 | 图形超出容器边界时不显示 |
| 1 | 嵌套裁剪 | 父子层级的裁剪关系 |
| 2 | 多层裁剪 | 多层级嵌套的裁剪效果 |
| 3 | 圆形裁剪 | 椭圆形裁剪区域 |
| 4 | 路径裁剪 | 复杂形状的裁剪 |
| 5 | 裁剪与事件 | 事件是否被裁剪边界限制 |
| 6 | 透明裁剪 | 半透明区域的裁剪效果 |
| 7 | 裁剪动画 | 动态变化的裁剪区域 |
| 8 | 裁剪性能 | 大量元素的裁剪效率 |
| 9 | 反向裁剪 | 挖空效果的裁剪 |

## 操作说明

- 按数字键 `0`-`9` 切换场景
- 按 `ESC` 退出程序

## 核心概念

### 裁剪原理

裁剪定义了图形的可见区域，超出该区域的部分不会渲染。

### 裁剪层级

裁剪会在层级中传递：

- 父容器的裁剪区域会影响所有子元素
- 子元素只能在父容器的裁剪区域内显示

## 依赖模块

- `novadraw-scene`: 场景图和 Figure 接口
- `novadraw-render`: Vello 渲染后端（含裁剪支持）
